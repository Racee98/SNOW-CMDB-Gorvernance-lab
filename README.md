# SNOW-CMDB-Gorvernance-lab
ServiceNow CMDB Governance Lab
Demonstrating enterprise-level CMDB governance, CI relationship mapping, configuration data management, and audit reporting aligned with ServiceNow CSDM and ITIL practices.

Business Scenario
A DoD contractor environment required traceable configuration baselines for a production SAP S/4HANA ERP deployment. The CMDB lacked structured service mapping, ownership enforcement, and health visibility — creating audit risk and slowing change approval cycles.

This lab implements a governance framework addressing all three gaps.

What This Project Demonstrates
CMDB CI lifecycle management across 4 CI classes
CSDM-aligned service dependency mapping
Bulk CI data onboarding via Import Sets and Transform Maps
Server-side data quality enforcement via Business Rules
CMDB health visibility through a multi-report dashboard
Architecture — CI Dependency Chain
